class sim_client.sim_client(name_file, remote_ip)¶

This class can be used to simulate client-server connection in case a smarthwatch is not available. In general sends messages as strings, then can be used for other types of projects.

Parameters:
  • name_file – txt file with with data taken of the smartwacth (wearAmi offline version)
  • remote_ip – ip of the server
Example: 
>>> from sim_client import*
>>> name_file = "prova.txt"
>>> remote_ip = "192.168.2.1
>>> c = sim_client(name_file,remote_ip)
class decode_message.decode_message(string)¶

This class decodes a message from a set of strings used in the wearAmi programs. Because there are different format it is necessary specify the delimiter.

Parameters:string – delimiter used. Example ”;”
decode(line)¶

Acceleration decodification

Parameters:
  • line – string with a sample of the accelerometer or gyroscope sensor
  • remote_ip – ip of the server
Returns:

  • success: True if the string has the correct format
  • option: if option == 0 then is aceleration data, if option = =1 then is gyroscope data
  • value1: x value
  • value2: y value
  • value3: z value

An example of a format used for a wearAmi program is:

  • <indicator>;number;x_value;y_value:z_value

where <indicator> can be a (acceleration) or y (gyroscope),
